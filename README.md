# Advent of Code 2025
Code for Mount Si Software's projects and public framework for Advent of Code 2025 (https://adventofcode.com/)

# Developer setup
* `npm install` should do the trick.
* If you want to use a port other than 3000, set PORT-<number> in a .env file
* `npm run start` runs the HTTP daemon process.
* `http://localhost:<port>/aoc2025` should bring up the main page.
